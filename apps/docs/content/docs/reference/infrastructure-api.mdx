---
title: Infrastructure API
description: Service APIs — email, queue, storage, KV, realtime, logger.
---

All services are available through handler context (routes, hooks, jobs).

## Email

```ts
// Send template
await email.sendTemplate({
  template: "templateName",
  input: { ... },
  to: "user@example.com",
});
```

## Queue

```ts
// Publish job
await queue.jobName.publish({
  // typed payload
});
```

## Storage

```ts
// Upload
const asset = await storage.upload(file);

// Get URL
const url = storage.getUrl(asset.path);
```

## KV Store

```ts
// Set with TTL
await kv.set("key", "value", { ttl: 3600 });

// Get
const value = await kv.get("key");

// Delete
await kv.delete("key");
```

## Realtime

```ts
// Broadcast
await realtime.broadcast("channel", { type: "event", data: { ... } });
```

## Logger

```ts
logger.info("message");
logger.error({ err: error }, "Error message");
logger.debug({ key: "value" }, "Debug info");
```

## Database

```ts
// Raw SQL
const result = await db.execute(sql`SELECT 1`);

// Drizzle query builder
const rows = await db.select().from(table).where(eq(table.id, id));
```

## App Instance

```ts
// Create context (for scripts)
const ctx = await app.createContext({ accessMode: "system", locale: "en" });

// Collection API via app
const posts = await app.api.collections.posts.find({}, ctx);
```
