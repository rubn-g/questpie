QUESTPIE uses [pg-boss](https://github.com/timgit/pg-boss) for background job processing. Jobs are stored in PostgreSQL — no external queue service needed.

## Configuration

```ts title="questpie.config.ts"
import { pgBossAdapter, runtimeConfig } from "questpie";

export default runtimeConfig({
	queue: {
		adapter: pgBossAdapter({
			connectionString: process.env.DATABASE_URL,
		}),
	},
});
```

## Publishing Jobs

From hooks, routes, or other jobs:

```ts
handler: async ({ queue }) => {
	await queue.sendAppointmentConfirmation.publish({
		appointmentId: "abc",
		customerId: "def",
	});
};
```

The `queue` object is fully typed — autocompletion shows all registered jobs and their payload schemas.
