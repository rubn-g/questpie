---
title: Hooks API
description: Lifecycle hooks reference — beforeValidate, beforeChange, afterChange, beforeDelete, afterDelete.
---

## Hook Definition

```ts
.hooks({
  beforeValidate?: async (ctx) => void,
  beforeChange?: async (ctx) => void,
  afterChange?: async (ctx) => void,
  beforeDelete?: async (ctx) => void,
  afterDelete?: async (ctx) => void,
})
```

## Hook Context

### `beforeValidate` / `beforeChange` / `afterChange`

| Property | Type | Description |
|---|---|---|
| `data` | `Record` | Record data (mutable in `before*`) |
| `operation` | `"create" \| "update"` | Current operation |
| `original` | `Record \| undefined` | Previous state (update only) |
| `collections` | `AppCollections` | Collection API |
| `globals` | `AppGlobals` | Globals API |
| `db` | `Database` | Drizzle instance |
| `session` | `Session \| null` | Auth session |
| `queue` | `QueueClient` | Job queue |
| `email` | `MailerService` | Email service |
| Custom services | — | Services from `services/` |

### `beforeDelete` / `afterDelete`

| Property | Type | Description |
|---|---|---|
| `id` | `string` | Record ID |
| `collections` | `AppCollections` | Collection API |
| `db` | `Database` | Drizzle instance |
| `session` | `Session \| null` | Auth session |
| `queue` | `QueueClient` | Job queue |

## Lifecycle Order

### Create / Update
```text
beforeValidate → Schema Validation → beforeChange → DB Write → afterChange
```

### Delete
```text
beforeDelete → DB Delete → afterDelete
```
