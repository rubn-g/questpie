Actions are operations users can perform on records — create, save, delete, duplicate, or custom operations with forms.

## How Actions Work

The admin panel provides built-in CRUD actions automatically. You can also define custom actions with forms and server-side handlers.

- [Built-in Actions](/docs/workspace/actions/builtin) — Create, save, delete, duplicate
- [Custom Actions](/docs/workspace/actions/custom) — Custom forms and handlers
