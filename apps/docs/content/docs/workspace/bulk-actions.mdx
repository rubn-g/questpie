The list view supports multi-select for batch operations. Select records with checkboxes, then use the floating toolbar to act on them.

## Using Bulk Actions

1. Open any collection list view
2. Check the checkbox on individual rows, or use the header checkbox to select all
3. A toolbar appears at the bottom with available actions
4. Click an action to apply it to all selected records

## Built-in Actions

| Action | Description |
|---|---|
| **Delete** | Delete all selected records. Prompts for confirmation. |

## Selection

- **Individual**: Click the checkbox on each row
- **Select all (page)**: Click the header checkbox to select all visible rows
- **Deselect**: Click the header checkbox again or close the toolbar

The toolbar shows the count of selected items.

## Soft Delete Support

If your collection has soft delete enabled, the bulk delete action soft-deletes records instead of permanently removing them. Soft-deleted records can be restored individually from the form view.
