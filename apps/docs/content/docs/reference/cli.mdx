## Commands

| Command                     | Description                             |
| --------------------------- | --------------------------------------- |
| `questpie generate`         | Scan file conventions, generate types   |
| `questpie dev`              | Watch mode — auto-regenerate on changes |
| `questpie push`             | Push schema to database (no migrations) |
| `questpie migrate:generate` | Generate migration from schema diff     |
| `questpie migrate:up`       | Run pending migrations                  |
| `questpie migrate:down`     | Rollback last migration                 |
| `questpie migrate:fresh`    | Drop all + re-run migrations            |
| `questpie migrate:reset`    | Reset migration tracking                |
| `questpie seed:run`         | Run seed files                          |
| `questpie add <module>`     | Add a module or starter                 |

## Usage

```bash
# With bunx
bunx questpie <command>

# With npx
npx questpie <command>
```

## Configuration

The CLI reads `questpie.config.ts` from the project root:

```ts title="questpie.config.ts"
export { default } from "./src/questpie/server/questpie.config";
```

## CLI Config Options

```ts
runtimeConfig({
	cli: {
		migrations: {
			directory: "./src/migrations",
		},
		seeds: {
			directory: "./src/seeds",
		},
	},
});
```
